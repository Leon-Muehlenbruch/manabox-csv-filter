# ManaBox CSV Filter

Browser-Tool zum Filtern von [ManaBox](https://manabox.app/) CSV-Exporten – z.B. für den Import in [Moxfield](https://moxfield.com/).

## Features

- **Drag & Drop** – CSV laden, Filter wählen, gefilterte CSV herunterladen
- **Konfigurierbare Filter:**
  - `Binder Type = list` entfernen
  - `Misprint = true` entfernen
- **Statistik** – zeigt wie viele Zeilen entfernt/behalten wurden
- **Komplett clientseitig** – keine Daten werden an einen Server gesendet

## Nutzung

1. In ManaBox: Collection → Menü (⋮) → Export → CSV
2. CSV auf [leon-muehlenbruch.github.io/manabox-csv-filter](https://leon-muehlenbruch.github.io/manabox-csv-filter/) laden
3. Filter konfigurieren → "Filtern & Download"
4. Gefilterte CSV in Moxfield importieren
